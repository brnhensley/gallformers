# Gall Admin (V2)

**Route**: `/admin/galls`
**Module**: `GallformersWeb.Admin.GallLive.Index`

## Summary

List all galls with CRUD operations. Form pages at `/admin/galls/new` and `/admin/galls/:id`. Includes browse functionality merged from V1's separate browse page. Additional undescribed galls management at `/admin/galls/undescribed`. LiveView-based.
