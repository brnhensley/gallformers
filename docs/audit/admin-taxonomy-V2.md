# Taxonomy Admin (V2)

**Route**: `/admin/taxonomy`
**Module**: `GallformersWeb.Admin.TaxonomyLive.Index`

## Summary

List and manage all taxonomic entries (families, genera, sections, species). CRUD operations with form pages at `/admin/taxonomy/new` and `/admin/taxonomy/:id`. LiveView-based.

**Note**: V2 consolidates Section management into this page (separate in V1).
