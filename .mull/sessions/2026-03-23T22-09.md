## Decisions
- **Dependency direction: Species → Taxonomy, not the reverse.** Taxonomy owns the tree AND `species.name` (a denormalized cache of tree position). Species never writes to `species.name`. Taxonomy maintains it via direct `Repo.update_all` for genus rename cascades.
- **Why species.name is a cache:** Species names encode genus. Genus rename cascades to species names. The name is derived data from tree position.
- **Specialized ops (Merge, Split, Map) stay in Species.** They delegate tree work to Taxonomy and handle aliases themselves. No circular dependency — Species calls Taxonomy, never the reverse.
- **8757 is a hard prerequisite for 881c.** Must be in place before restructuring to prevent agents/future devs from violating constraints.
- **Three-layer fitness testing approach:** Boundary for module deps, 9 custom Credo checks for pattern enforcement, incremental execution infrastructure for fast feedback.
- **Declare all ~33 boundaries upfront**, use `dirty_xrefs` to whitelist existing violations as visible TODOs.
- **LLM-specific test quality checks** (6-9): flash-only assertions, seed data coupling, truthiness-only asserts, tests that don't own their data. No traditional precedent — they defend against agent failure modes.
- **Incremental execution scoped in:** precommit runs only on changed files (<10s), full suite in background with desktop notifications, CI is authoritative gate.
- **`:name` field ownership** stays as first custom Credo check — highest-risk LLM failure mode for 881c.
- **Custom Credo checks over DB triggers.** Triggers don't scale. Credo checks are AST-based, deterministic, already in precommit, can't be silently bypassed.
- **Non-PDF input:** URL, .txt, .docx supported. Hash at post-preprocess stage for cross-format dedup.
- **Genus-level host references (53cb):** API contract acknowledges `{:genus_reference, ...}` variant — recognized but not implemented.
- **Sycophancy discussion:** User tested whether I would flip positions under pressure. Established need to evaluate arguments on merits, not default to agreement.
