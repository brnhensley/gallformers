# Admin workflow improvements

## Survey Feedback (2026-03-04)

"For adding galls and other admin tasks, a guided (prompts? enforced?) step by step process would be helpful so things aren't missed or forgotten. Currently each step feels rather isolated and the user isn't prompted to 'start here' and then complete other necessary or helpful steps."

## Ideas

- Wizard-style flow for adding new galls (create species → add traits → link hosts → add sources → add images)
- Checklist/progress indicator showing what's been completed vs. missing
- "Next step" prompts after completing each admin action
- Data completeness indicator on admin pages (already have `data_complete_badge` component — could expand)
